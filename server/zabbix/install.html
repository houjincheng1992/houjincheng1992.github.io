          # zabbix 安装

          zabbix 整体分为两个部分 agent 与 server:

          1. zabbix agent需要安装在被监视的目标服务器上，它主要完成对硬件信息或与操作系统有关的内存，CPU等信息的收集。

          2. zabbix server可以单独监视远程服务器的服务状态；同时也可以与zabbix agent配合，可以轮询zabbix agent主动接收监视数据（agent方式），同时还可被动接收zabbix agent发送的数据（trapping方式）。

          # 环境

          名称 版本
          系统 centos 7
          zabbix版本 4.0LTS
          MySQL 5.7.27

          # zabbix server 的安装

          1. Install Zabbix repository
          # rpm -Uvh https://repo.zabbix.com/zabbix/4.0/rhel/7/x86_64/zabbix-release-4.0-2.el7.noarch.rpm
# yum clean all
          1. Install Zabbix server, frontend
          # yum -y install zabbix-server-mysql zabbix-web-mysql
          1. Create initial database
          # mysql -uroot -p
password
mysql> create database zabbix character set utf8 collate utf8_bin;
mysql> grant all privileges on zabbix.* to zabbix@localhost identified by 'password';
mysql> quit;

          备注 此处需要将password换成相对应的符合密码规范的密码

          1. 导入zabbix所需的sql
          # zcat /usr/share/doc/zabbix-server-mysql*/create.sql.gz | mysql -uzabbix -p zabbix
          1. Configure the database for Zabbix server

          Edit file /etc/zabbix/zabbix_server.conf

          DBPassword=password

          备注 此处需要将password换成刚才替换的密码

          1. Configure PHP for Zabbix frontend

          Edit file /etc/httpd/conf.d/zabbix.conf, uncomment and set the right timezone for you.

          # php_value date.timezone Asia/Shanghai
          1. Start Zabbix server processes

          Start Zabbix server and agent processes and make it start at system boot:

          # systemctl restart zabbix-server httpd
# systemctl enable zabbix-server httpd

          备注 可能中间会出现一些错误，可能是部分listenport未打开，参考agent中打开端口的方式，尝试打开10051端口。

          1. frontend 操作
          参考文档
https://www.zabbix.com/documentation/4.0/manual/installation/install#installing_frontend

          # zabbix agent 的安装

          1. Install Zabbix repository
          # rpm -Uvh https://repo.zabbix.com/zabbix/4.0/rhel/7/x86_64/zabbix-release-4.0-2.el7.noarch.rpm
# yum clean all
          1. Install Zabbix agent
          # yum -y install zabbix-agent

          备注 此处可能会有失败，主要是openssl与zabbix-agent的不匹配的问题

          1. 添加与查看防火墙接口
          # 查看指定区域所有打开的端口
# firewall-cmd --zone=public --list-ports

# 在指定区域打开端口 (需要重启防火墙)
# firewall-cmd --zone=public --add-port=10050/tcp --permanent

# 重启防火墙
# firewall-cmd --reload
          1. Start Zabbix agent processes
          # systemctl restart zabbix-agent
# systemctl enable zabbix-agent
          1. 相关配置修改

          需要修改/etc/zabbix/zabbix_agentd.conf

          # Passive checks related - Server
Server = server地址

# Active checks related
ServerActive = server地址
          1. 重启agent
          # systemctl restart zabbix-agent
